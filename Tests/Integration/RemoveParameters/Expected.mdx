## SYNOPSIS

Dummy module to test the -RemoveParameters switch

## SYNTAX

```powershell
Test-RemoveParameters [[-KeepThisOne] <String>] [[-RemoveMe] <String>] [[-AlsoKeep] <String>]
 [[-HideMe] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```
