## SYNOPSIS

Test function for PlatyPSMarkdownPath parameter.

## SYNTAX

```powershell
Test-CachedMarkdown [[-CachedParam] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

This function tests using pre-generated PlatyPS markdown files instead of generating them from a module.
